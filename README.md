# AI Prompt Generator Pro

A premium AI‑powered web application that converts images and videos into highly detailed, professional prompts ready for AI image generators like **Midjourney**, **DALL‑E**, and **Stable Diffusion**.

## Features

- **Image to Prompt Generator**
  - Upload any image (JPG, PNG, WEBP).
  - Performs deep visual analysis: subject, lighting, colors, textures, mood, and more.
  - Generates a fully descriptive English prompt (80+ words).
  - Outputs a strict JSON object with 10 analytical fields.
  - Supports aspect ratio selection (1:1, 16:9, 9:16, etc.).

- **Video to Prompt Generator (Segmented Cinematic)**
  - Upload a video (MP4, MOV, WEBM).
  - Automatically extracts frames at evenly spaced time intervals.
  - Analyzes each frame individually and creates a unique cinematic prompt per scene.
  - Customizable number of segments (3–8).
  - Displays a storyboard grid with thumbnails and individual copy buttons.
  - Exports a complete JSON array of all scenes.

- **Professional output**
  - All prompts are production‑ready and include **8K, ultra‑detailed, hyperrealistic** quality tags.
  - Copy any prompt with one click.
  - Download full JSON analysis as a `.json` file.
